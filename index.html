Frequently Asked Questions

what is difference between var, let, and const ?
var: Function/global scoped (not block scoped).Can be re-declared and updated in the same scope.
let: Block scoped (lives inside {}).Can be updated but not re-declared in the same block.
const: Block scoped.Cannot be updated or re-declared.
Whats difference between map(), forEach(), and filter() ?
forEach() just runs a function on every item in an array, but doesn't return anything.
map() runs a function on every item and gives you back a brand-new array with the results.
filter() goes through each item and only keeps the ones that pass a condition, giving you a new array.
what are arrow functions and how they are different from regular functions?
arrow functions are shorter way to write a function in JavaScript.
Differences Between Arrow Functions and Regular Functions:
"this" behavior: Regular functions have their own this (depends on how the function is called). Arrow functions do not have their own this; they inherit this from the parent scope (lexical scoping).
Syntax: Arrow functions have a shorter, cleaner syntax, especially useful for one-liners. Regular functions are longer and can be hoisted (used before declared).
How JavaScript Promises work?
A Promise is like a placeholder for something that will finish later.It's either pending,success or error. "promise" to do something later and handle it with .then() or .catch().
How does closures work in JS ?
A closure happens when a function remembers variables from where it was created, even after that outer function is done running. It's like the inner function has a backpack, carrying those variables around.
